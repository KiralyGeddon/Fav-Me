Fav-Me - Votre Gestionnaire de Favoris Personnel

Bienvenue sur Fav-Me !

Fav-Me est une application de bureau conçue pour vous aider à organiser et à accéder rapidement à vos dossiers et sites web favoris. Fini les recherches fastidieuses, regroupez tout ce qui est important en un seul endroit pratique.

--- Fonctionnalités Clés ---

* Gestion facile des dossiers favoris :
    - Ajoutez n'importe quel dossier de votre ordinateur.
    - Ouvrez-les directement en un clic.
    - Modifiez ou supprimez-les à tout moment.
    - Fav-Me vérifie automatiquement si vos dossiers existent toujours sur votre système.

* Gestion des sites web favoris :
    - Enregistrez vos adresses web préférées.
    - Ouvrez-les dans votre navigateur par défaut.
    - Fav-Me tente même d'afficher l'icône (favicon) de chaque site web pour une reconnaissance visuelle rapide.
    - Modifiez ou supprimez vos sites web favoris.

* Affichage personnalisable :
    - Basculez facilement entre la vue de vos dossiers et celle de vos sites web favoris.
    - Personnalisez l'apparence de l'application : choisissez entre les modes sombre, clair ou système, et sélectionnez votre couleur préférée (thème).

* Vos données sont sauvegardées :
    - Tous vos favoris et vos paramètres de thème sont enregistrés automatiquement. Vous retrouverez toujours votre application telle que vous l'avez laissée.

--- Comment Utiliser Fav-Me ---

1.  Ajouter un favori : Cliquez sur le bouton "Ajouter un favori". Une fenêtre vous demandera si vous souhaitez ajouter un dossier ou un site web, puis de saisir les détails.
2.  Changer de vue : Utilisez le bouton en haut à gauche pour passer de l'affichage des dossiers à celui des sites web, et vice-versa.
3.  Ouvrir un favori : Un simple clic sur le nom d'un dossier ou d'un site web l'ouvrira immédiatement.
4.  Modifier ou Supprimer : Pour modifier ou supprimer un favori, utilisez les icônes crayon (modifier) et corbeille (supprimer) à côté de chaque élément.
5.  Paramètres de l'application : Le bouton en haut à droite (icône d'engrenage) ouvre les options pour changer le mode d'apparence et le thème de couleur de Fav-Me.

--- Où sont stockées vos données ? ---

Vos favoris et vos paramètres sont stockés dans des fichiers appelés "favorites_config.json" et "app_settings.json". Ces fichiers se trouvent dans un dossier dédié à l'application sur votre système :

* Sous Windows : Dans le dossier AppData de votre utilisateur, généralement à cet emplacement :
    C:\Users\VotreNomUtilisateur\AppData\Local\FavMeData\
* Sous macOS : Dans le dossier Application Support de votre bibliothèque utilisateur :
    ~/Library/Application Support/FavMeData/
* Sous Linux : Dans le dossier de configuration :
    ~/.config/FavMeData/
